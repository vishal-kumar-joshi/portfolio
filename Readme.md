# 🌐 Personal Portfolio Website

A modern, responsive and visually appealing personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, and journey as a developer.
 
---

## 👨‍💻 About Me

I am a passionate **Python Developer** and BCA student with a strong interest in software development, problem solving, and building real-world applications.

I enjoy learning new technologies and continuously improving my skills in backend development, AI/ML, and system design.

---

## 🚀 Features

- Fully responsive design (mobile + desktop friendly)
- Modern UI with smooth animations
- Interactive navigation menu (mobile sidebar)
- About section with skills, education & experience tabs
- Projects showcase section
- Image gallery with fullscreen viewer
- Contact form integrated with Google Sheets
- Social media integration
